What are You Doing?

You're here because you want to see a real live application written in Python and interacting with the Amazon Flexible Payment Service via the FPyS library. You probably want to download the source.

If you are here for any other reason, you are probably in the wrong place.

How Does it Work?

This demo lets you "buy" the ability to view full sized versions of the thumbnails to the left. Of course, this demo is wired to the Amazon FPS Sandbox, so no money actually changes hands, even if you do complete an order. Heck, the full size images aren't even really protected. It's all a charade in the name of education.
